\doxysubsection{Detailed Description}
\begin{DoxyVerb}    Small wsgiref based web server. Takes a path to serve from and an
    optional port number (defaults to 8000), then tries to serve files.

    Mime types are guessed from the file names, 404 errors are raised
    if the file is not found. Used for the make serve target in Doc.

    Restarts if any of the files in current dir changed and project dir
    changed; calls the bringfocus batch job to refresh browser\end{DoxyVerb}
